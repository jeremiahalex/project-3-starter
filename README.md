## To work on this.

1. Fork and clone this repo
2. ```bundle install```
3. Set up Stripe API (sign up and input the keys to you .env file)
4. Add your email account username and password in .env file
5. ```rails db:create```
6. ```rails db:migrate```
7. ```rails s```
